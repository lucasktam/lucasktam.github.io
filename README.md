# lucasktam.github.io

Hello! My name is Lucas Tam. I study Computer Science at the University of Michigan in Ann Arbor.

Recently, I completed a scalable game engine as a part of EECS 498 - Game Engine Architecture at UofM. 
It's built on C++, Lua, GLM, LuaBridge, SDL, and Box2D. 
I also added a custom feature - a custom 2d procedural terrain generator - that runs on Python, numpy, matplotlib, tkinter, lua, and scipy.
